# Simplex method calculator

Solve the minimization problem by simplex method for:
```
ax_1 + ... + ax_n -> min  
ax_1 + ... + ax_n [<=, <, =] z (1)
...
ax_1_p + ... + ax_n_p [<=, <, =] z_p (p)
p = 1..5, n = 1..5
```
